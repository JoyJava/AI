### 生成报告前必须执行

1. **用工具验证关键数字** — 在写入任何具体仓位数字、AUM、现金余额前：
   - 巴菲特：读取最新 10-Q/10-K（berkshirehathaway.com/qtrly/）
   - 德鲁肯米勒/伯里/阿克曼等：查最新 13F（SEC EDGAR 或 13f.info / insidermonkey.com）
   - 阿申布伦纳：查 Insider Monkey 或 HedgeFollow
   - 都铎·琼斯/达利欧：标注为"推算"，不给精确数字

2. **不编造超出公开数据范围的数字** — 如果某季度 13F 尚未公开（申报截止前），必须：
   - 明确标注"⚠️ 推算值，待 XX 日后 13F 公开验证"
   - 不得给出精确百分比，只能说"预计延续 X 趋势"

3. **已知数据缺失必须声明**：
   - Scion AM（伯里）：2025年11月10日注销投资顾问资格，Q3 2025 为最后一份 13F。此后无公开仓位数据，不得编造
   - 宏观基金（Bridgewater/Tudor）：13F 仅反映美股多头权益，期货/外汇/商品仓位不在其中，标注"仅为组合一角"
   - 家族办公室：德鲁肯米勒 13F 仅 ~$4B 美股部分，不含非美/大宗/外汇

4. **13F 的固有局限（每次报告必须提醒用户）**：
   - 空头/Put 的报告方式是"名义市值"≠实际投入本金
   - 45天滞后：报告看到的是 1.5 个月前的快照
   - 不含非美资产、期货、互换、私募投资

## 核心认知：绝对仓位数字没有信号价值

```
❌ 错误用法：格兰瑟姆仓位只有20%，说明他很看空
✅ 正确用法：格兰瑟姆仓位一直只有20%，这是他的基因，不是信号

真正有用的四个维度：
  ① 变化速度     — 单季度变化 >10% 才是强信号
  ② 跨风格共识   — 价值派+宏观派+激进维权派同时减仓
  ③ 资金去向     — 短债=等待；黄金=结构担忧；做空期权=最强看跌
  ④ 行动vs语言  — 有代价的行动 > 无代价的言论
```

## 逃顶指标体系（三层结构）

### 第一层：估值泡沫（有没有泡沫？）

| 指标                          | 危险线 | 说明                                           |
| ----------------------------- | ------ | ---------------------------------------------- |
| Buffett Indicator（市值/GDP） | >135%  | 生成时查 currentmarketvaluation.com 获取实时值 |
| CAPE 席勒市盈率               | >30    | 历史上只在1929/2000/2020+出现                  |
| Excess CAPE Yield             | <0%    | =1/CAPE - 10年国债收益率，负值=股票无超额回报  |
| P/S 市销率                    | >2.5x  | 比P/E更难被盈利操纵                            |

### 第二层：流动性/信用（钱还在不在？）

| 指标                        | 危险线                     | 说明                      |
| --------------------------- | -------------------------- | ------------------------- |
| 高收益信用利差（HY Spread） | >500bp                     | 信用市场领先股市6-12月    |
| 10Y-2Y 收益率曲线           | 倒挂后**恢复正常**时最危险 | 恢复后衰退通常在几月内    |
| M2 货币供应增速             | <0%（负增长）              | 2022年出现，6月后股市深跌 |
| 美联储降息开始              | 第一次降息后               | "降息不是底部，是哨声"    |
| 银行放贷标准 SLOOS          | 净收紧>40%                 | 与衰退高度相关            |

### 第三层：市场结构/情绪（是否开始定价？）

| 指标                    | 危险线                     | 说明                          |
| ----------------------- | -------------------------- | ----------------------------- |
| A/D Line 涨跌线背离     | 指数新高但A/D不跟          | 上涨股票占比<50%=权重股撑着   |
| VIX 期限结构            | 近月>远月（Backwardation） | 近端恐慌激增，聪明钱买保险    |
| SKEW Index              | >145 持续出现              | VIX低+SKEW高=表面平静暗流涌动 |
| AAII 牛市情绪           | >55%                       | 散户极度乐观=接盘侠耗尽       |
| 融资余额（Margin Debt） | 同比增速>60%               | 杠杆极端化                    |
| 内部人士减持            | 净卖出/净买入<0.3          | 最了解公司的人在逃跑          |
| 板块轮动                | XLU/XLK 比值上升           | 资金悄悄从科技转向公用事业    |
| 铜金比                  | 持续下降                   | 经济预期恶化的领先信号        |
| IPO 数量                | 无盈利IPO占比激增          | 流动性泡沫顶部特征            |
| 股票回购                | 高管大量减持+回购停止      | 最懂公司的人在卖              |

### 综合决策阈值

```
亮灯数量（21个指标）→ 对应行动：
0-2个：正常，不行动
3-4个：开始减仓（每月减10-15%）
5-7个：显著减仓至50-60%
8-10个：大幅减仓至30-40%
>10个：极度防御，10-20%仓位
```

## 仓位信号四维实操（每季度20分钟）

**第一步（5分钟）：筛变化速度**
- 工具：Dataroma.com 或 SEC EDGAR
- 阈值：>5% 记录 / >10% 强信号 / 多人同季>10% 极强信号

**第二步（5分钟）：看跨风格共识**
- 强信号：价值派（巴菲特）+ 宏观派（德鲁肯米勒）+ 激进维权（阿克曼）同时减仓
- 弱信号（忽略）：格兰瑟姆减仓（他一直在减，是基因）

**第三步（5分钟）：看资金去向**
- 短债/货币基金 = 等待机会，随时可回来
- 黄金/实物资产 = 结构性担忧，偏悲观
- 做空期权/CDS = 最强看跌，有实际代价

**第四步（5分钟）：交叉验证宏观**
- 五信号同时亮灯才行动：大佬加速减仓 + 高收益利差扩张 + 市场广度恶化 + VIX结构上移 + 收益率曲线倒挂后恢复

### 减仓顺序

```
第一批减（最先卖）：
  高估值成长股（P/S>15x，无盈利）
  高杠杆公司（债务/EBITDA>5x）
  小盘股（流动性差）
  新兴市场ETF

第二批减：
  周期性行业（金融/工业/材料）
  高Beta股票

最后保留/置换防御：
  高股息低波动（必需消费/公用事业）
  短期美国国债（T-Bills，1-3个月）
  黄金ETF（GLD/IAU）
```

### 防御资产优先级

| 资产                  | 优先级 | 适用场景                 |
| --------------------- | ------ | ------------------------ |
| 1-3月T-Bills          | ⭐⭐⭐    | 任何防御情景             |

### 三个执行纪律

```
1. 写下来：在平静时制定计划，动荡时只执行文档
2. 月度检查：不做日内交易，每月看一次指标状态
3. 接受"早了一点"：减仓后市场继续涨，不要追回去
4. 不懂的不要编故事, 可以告诉我"我不知道"，但不要编造
```

## 仓位分子/分母说明

```
仓位占比 = 13F披露权益多头市值 / 基金总AUM

陷阱：
  ① 对冲基金空头不在13F内 → 多头仓位≠净多头
  ② 宏观基金期货/互换/外汇不在13F内 → 13F仅反映美股权益部分
  ③ 13F有45天滞后，大佬可能已完全反转
  ④ 13F名义价值 ≠ 实际本金（Put报告金额是底层资产市值，非期权费）
  ⑤ 已退出：伯里2025Q4起不再提交（注销），此后无法跟踪

德鲁肯米勒特别说明：
  - Duquesne 持续提交13F（管理>$1亿美股权益，具体AUM查最新13F）
  - 但13F仅反映美股权益部分，不含黄金/大宗/外汇/非美资产
  - 他的黄金/BTC观点来自CNBC采访（2024/10、2025/05/06），而非13F持仓
  - 近期13F中无GLD/IAU等黄金ETF持仓
  - 将采访观点(黄金/BTC)与13F持仓(美股权益)混为一谈是常见错误

最可信：巴菲特(10-Q/10-K)、阿克曼(PSH季报)、凯茜·伍德(ETF每日)、阿申布伦纳(13F)
部分可信：德鲁肯米勒(13F可查但仅一角)、特珀(13F可查)
最不可信：达利欧(风险平价)、都铎·琼斯(宏观期货)、马克斯(信用基金)
```

```
降仓主力动作完成时间（方向性结论，具体数字需实时查）：
  2019-2020：峰值期，多数大佬维持高仓位
  2021-2022：主力降仓期（美联储加息预期+实际加息），完成大部分降仓动作
  2023至今：微调/等待期，季度变化趋小

近年变化小的定性解释：
  主要降仓在2021-2022年完成，不是他们改变了看法，
  而是仓位已降到各自目标区间。

⚠️ 上述为基于历史观察的定性叙述。
   报告中如需引用"平均仓位"数字，必须实时计算：
   逐个查询最新13F → 计算可比组均值 → 标注计算日期和口径。
```

## 框架局限性

```
大佬仓位能告诉你：
  ✅ 聪明钱的风险偏好方向
  ✅ 峰值距离（还有多少仓位可以继续减）
  ✅ 共识形成程度

大佬仓位不能告诉你：
  ❌ 市场什么时候跌
  ❌ 跌多少
  ❌ 你应该做什么（税务/流动性约束不同）

这是情绪温度计，不是时钟。
单独看任何一个指标都是噪音。
五个系统同时发信号时才是行动窗口。
```

## 生成HTML报告

输出：`{2026-05-23}大佬仓位追踪.html`

报告包含：
- 四维框架卡片（变化速度/跨风格共识/资金去向/行动vs语言）
- 1990Q1-2026Q1 仓位历史折线图（Chart.js，每季度一个点，锚点+插值）
- 仓位信号表（含历史峰值、过去 4 个季度的仓位占比（当时市值/当时资金总额）以及当前仓位在历史 5 年中的百分位）
- 历史崩盘预测成功率排名（S/A/B/C级）
- 实操手册（每季度20分钟检查流程）
- 投资人简介
