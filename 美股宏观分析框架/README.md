# 美股宏观分析框架（Claude Code Skill）

> 一套用于追踪美国宏观经济的**消费者现金流传导链**框架，专为 [Claude Code](https://claude.com/claude-code) 设计。
> 核心价值：识别 **K 型分化** 和 **幸存者偏差**，提前 12-24 个月预警系统性风险。

---

## 这是什么

这个 skill 把一套**11 阶段传导链分析法**固化成 Claude Code 可自动执行的流程。  
当你对 Claude 说"最近美股宏观怎么样"或"我该减仓吗"，它会：

1. 依次检查 11 个阶段（收入→储蓄→举债→消费降级→信用→踢出→硬资产→破产→银行→实体→市场）
2. 为每个阶段独立判断信号灯颜色（🟢🟡🟠🔴）
3. 对比当前与 **1995-2005 dotcom 周期** 和 **2003-2008 次贷周期** 的相同位置
4. 生成一份综合诊断 HTML 报告

## 为什么这套框架

- **抗幸存者偏差**：传统信用卡逾期率在 2007 年看起来很健康，因为银行正在砍次级发卡——分母缩小制造了假象。本框架用**绝对金额 + 次级专营商独立指标**绕开。
- **看分层不看平均**：储蓄率被顶层 10% 拉高时，底层 50% 可能已经-2%。始终看 SF Fed 分层数据。
- **看财报原文不看媒体**：消费降级的早期信号藏在 WMT/DG/MCD 的 10-Q 和 earnings call 里，不在 CNBC 头条里。
- **多年序列对比**：单点值（"2008 是 X"）丢失轨迹信息。本框架用 1995-2005 和 2003-2008 **每年一个值**的完整序列对比，让读者看出"当前位于历史曲线哪一段"。

## 快速开始

### 安装

**选项 A：全局安装（推荐）**

```bash
# 克隆到任意位置
git clone https://github.com/YOUR_USERNAME/us-macro-skill.git ~/projects/us-macro-skill

# 符号链接到 Claude Code skills 目录
ln -s ~/projects/us-macro-skill ~/.claude/skills/美股宏观分析框架
```

**选项 B：项目级使用**

直接把整个目录放在你的 Claude Code 项目根目录下，Claude 会自动发现 `.skill` 后缀的目录。

### 使用

1. 在 Claude Code 中打开任意项目
2. 输入触发性问题：
   - "最近美股宏观怎么样？"
   - "我应该减仓吗？"
   - "K 型分化到什么程度了？"
3. Claude 会自动调用本 skill，按 11 阶段产出诊断 HTML

### 设置周度自动执行（可选）

在 Claude Code 中请求：

```
帮我设置每周五上午 9 点自动运行美股宏观诊断，
报告放到 投资/分析结论/ 目录
```

Claude 会用 `CronCreate` 创建周度任务。

## 核心设计原则

| 原则 | 说明 |
|------|------|
| **多年序列优于单点** | 每个指标给 5-10 年连续值，让读者看"轨迹位置" |
| **领先指标优于滞后** | JOLTS 辞职率、次级准入率 > 失业率、信用卡逾期率 |
| **分层指标优于平均** | 底层 20% 工资、底层 50% 超额储蓄 > 总体平均数 |
| **交叉验证** | 多职工作者 + 非经济性兼职；NCO + 汽车收回量 |
| **来源分级** | 🏛️ = SEC EDGAR 原文；✅ = 官方统计；⚠️ = 估算 |

## 框架局限性（坦白讲）

**擅长**：信用驱动型经济放缓（2008 型）、K 型分化识别、12-24 个月提前预警

**不擅长**：
- 资产泡沫型崩盘（2000 型）—— 需补充估值极端预警
- 外部冲击（2020 型）—— 无法预测
- 精确择时 —— 只给"阶段"，不给月份

**使用原则：框架给方向，不给精确时点。分批操作，不赌单一时刻。**
