# 决策规则

## 一、核心决策逻辑

**看传导链"压力到了哪一步" + "市场定价了没有"**

```
决策 = f(压力位置, 市场反应)

如果 压力在前端(1-4) + 市场没反应 → 开始减仓（提前跑）
如果 压力在中段(5-7) + 市场没反应 → 显著减仓（窗口期）
如果 压力到后端(8-10) + 市场没反应 → 大幅减仓（最后机会）← 当前
如果 压力到后端 + 市场开始反应(11) → 已经晚了但还能跑
如果 全线爆发 + 市场暴跌 → 防御/等待底部信号
```

## 四、特殊场景决策

### 4.1 K型分化加剧场景

```
特征：低收入全面恶化（阶段1-8红灯），但高收入仍然强劲
      → 表现为：S&P 500靠科技股撑着，整体指数不跌

应对：
  ├─ 减持：小盘股、消费必需品（Dollar General等）、区域银行
  ├─ 观望：大盘科技股（可能最后才跌）
  └─ 增持：防御性资产
```

### 4.2 流动性危机场景

```
特征：信用利差突然暴涨（1周内扩大200bp+），VIX跳涨至40+
      → 2020年3月式的恐慌

应对：
  ├─ 不要恐慌性清仓（最低点通常在VIX峰值附近）
  ├─ 如果之前已按计划减仓到50-60% → 等待
  ├─ 观察美联储是否紧急干预（降息、QE）
  └─ 美联储干预后 → 反而可能是加仓时机
```

### 4.3 政策救市场景

```
特征：美联储大幅降息或重启QE

应对：
  ├─ 短期（1-3月）：市场可能反弹
  ├─ 但基本面恶化（阶段1-8）不会立刻改善
  ├─ 政策生效有6-12个月时滞
  └─ 可小幅加仓10-15%，但不要全面进攻
```

## 六、决策执行纪律

1. **按计划执行，不感情用事。** 市场不跌时，减仓会感觉"傻"。但框架说减就减。
2. **分批执行。** 不要一次性清仓或加仓。每次操作不超过总仓位的10-15%。
3. **每月更新一次追踪表。** 只在月度更新时做决策，不做日内交易。
4. **触发器触发时，24小时内执行。** 触发器是例外，出现时不要等月底。
5. **记录每次决策的理由。** 方便复盘。
