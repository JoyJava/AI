# 指标详解：分子/分母拆解与扭曲风险

> 每个指标的完整拆解，确保不遗漏"系统外"人群。

---

## 数据格式约定（诊断报告生成时必读）

### 对比列格式
每个指标在 HTML 报告的对比表中**不要**写成单点值（如 "2000: 4.0%"），必须写**多年序列**，每年一个值：

```
1995~2005：1.5 1.6 1.8 2.1 2.4 2.7 2.5 2.3 2.0 2.2 2.4
2003~2008：2.4 2.5 2.3 1.7 1.4 1.2（从繁荣滑向崩盘）
2025~现在：2.1 1.9
```

**为什么多年序列**：单点值会丢失"轨迹信息"——从 2.4% 降到 1.4% 用了 5 年还是 1 年，结论完全不同。多年序列让读者看出当前位置在历史曲线的哪一段。

### 单位约定
- 比率类（失业率、储蓄率、逾期率）：百分比 `%`
- 金额类（信用卡余额、GDP）：十亿美元 `$B` 或万亿 `$T`
- 数量类（破产、收回量）：年化绝对数 `万件`

### 小数点精度
- 比率保留 1 位小数（`4.1%` 而非 `4.08%`）
- 除非指标本身精度要求更高（VIX 指数保留整数）

### 来源标注
- 🏛️ = 来自 SEC EDGAR 10-K/10-Q/8-K 原文
- ✅ = 官方统计（FRED/BLS/BEA/Fed 等）已核实
- ⚠️ = 估算 / 行业协会 / 无法核实

## 阶段1：收入端压力

### 1.1 实际工资增速

```
公式：名义小时工资增速 − CPI通胀率

分子（名义工资）受以下影响：
  ├─ 劳动力供需（移民政策、退休潮）
  ├─ 行业结构变化（高薪科技 vs 低薪服务业）
  └─ 最低工资法规

分母（CPI通胀率）受以下影响：
  ├─ 能源价格（油价）
  ├─ 住房成本（租金 OER 权重最大）
  └─ 食品价格
```

**扭曲风险：**
- 平均数被高收入拉高 → 必须看底层20%分位数
- CPI权重不反映低收入人群的支出结构（他们食品和能源占比更高）
- 亚特兰大联储的 Wage Growth Tracker 按分位数提供数据，优于BLS平均数

**数据源链接：**
- BLS: https://www.bls.gov/news.release/empsit.toc.htm
- 亚特兰大联储: https://www.atlantafed.org/chcs/wage-growth-tracker

---

### 1.2 多职工作者

```
公式：持有2份以上工作的人数 ÷ 总就业人数

分子（多职工作者）受以下影响：
  ├─ 单份工资不够生活 → 被迫打两份工
  ├─ 零工经济（gig economy）普及 → 主动兼职
  └─ 统计方法变化（零工是否被计入）

分母（总就业人数）受以下影响：
  ├─ 人口增长
  ├─ 劳动参与率变化
  └─ 季节性因素
```

**扭曲风险：**
- 零工经济增长可能推高数字（不完全是"被迫"）
- 需结合"因经济原因做兼职"（involuntary part-time）人数一起看

---

### 1.3 辞职率 / 招聘率（JOLTS）

```
辞职率 = 月度主动辞职数 ÷ 总就业
招聘率 = 月度新招聘数 ÷ 总就业

辞职率下降的含义：
  ├─ 工人不敢换工作 → 对就业市场缺乏信心
  ├─ 新机会减少 → 企业招聘放缓
  └─ 领先失业率上升 6-12 个月

招聘率下降的含义：
  ├─ 企业需求萎缩
  ├─ 招聘冻结（hiring freeze）蔓延
  └─ 领先失业率上升 3-6 个月
```

**扭曲风险：**
- JOLTS数据有1-2个月滞后
- 某些行业（科技）裁员可能被其他行业（医疗）招聘掩盖 → 需看行业拆分

## 阶段2：储蓄耗尽

### 2.1 个人储蓄率

```
公式：个人储蓄 ÷ 可支配个人收入

个人储蓄 = 可支配个人收入 − 个人消费支出

分子（储蓄）受以下影响：
  ├─ 收入变化
  ├─ 消费支出变化
  └─ 利息支出（信用卡利息在增加）

分母（可支配收入）受以下影响：
  ├─ 税收政策
  ├─ 政府转移支付（刺激支票等）
  └─ 社保扣除
```

**⚠️ 关键扭曲：**
- 这是**平均数**，被高收入严重拉高
- 顶层10%的储蓄率可能是20%+，底层50%可能是-2%
- **必须看旧金山联储的分层超额储蓄数据**

### 2.2 分层超额储蓄

```
公式：各收入分位的累计储蓄 − 疫情前趋势线

超额储蓄变化受以下影响：
  ├─ 消费恢复速度（底层花得更快）
  ├─ 通胀侵蚀（底层被侵蚀更多）
  ├─ 资产增值（股票/房产只帮到有资产的人）
  └─ 政府补贴到期
```

**数据源链接：**
- 旧金山联储: https://www.frbsf.org/research-and-insights/data-and-indicators/pandemic-era-excess-savings/

## 阶段3：举债阶段

### 3.1 循环信贷（信用卡余额）

```
公式：信用卡总余额月度变化

增长 → 可能含义：
  ├─ 消费者在借钱维持生活（坏）
  └─ 消费信心强、提前消费（好）
  → 需结合逾期率判断是"主动借"还是"被迫借"

增速下降 → 可能含义：
  ├─ 消费者主动去杠杆（好）
  └─ 银行收紧发卡、借不到了（坏）
  → 需结合次级发卡率（阶段6）判断
```

**⚠️ 关键扭曲：**
- 只包含"系统内"借款人
- 被银行拒卡的人 → 转向BNPL、EWA → 不在G.19数据中

### 3.2 BNPL 食品杂货占比

```
逻辑：BNPL 最初用于买电子产品/服装（非必需品）
      当人们用 BNPL 买食品杂货时 → 说明现金流已经撑不住基本生活

公式：BNPL交易中食品/杂货类占比

占比上升的驱动：
  ├─ 工资不够覆盖基本支出
  ├─ 信用卡已刷满或被砍额度
  └─ 储蓄耗尽
```

**数据来源：** Affirm (AFRM) 季度财报中的 GMV by category

### 3.3 提前领薪（EWA）

```
逻辑：人们付费提前拿到自己的工资
      → 说明一个月的现金流周转不过来
      → 是"发薪日效应"的数字化证据

市场规模快速增长 = 越来越多人月底没钱
```

**扭曲风险：**
- 市场增长部分是因为产品渗透率提升（不纯粹是压力驱动）
- 但增速>50%很难纯粹用渗透解释

## 阶段4：消费降级

### 4.1 消费降级阶梯详解

```
降级顺序（基于边际效用递减理论）：
  阶段4a 砍奢侈品 → 最不必要，第一个被砍
  阶段4b 砍外食 → 在家做饭替代
  阶段4c 砍家居/装修 → 可推迟
  阶段4d 降级品牌 → 同类替换到更便宜的
  阶段4e 降级业态 → 换到更便宜的商店
  阶段4f 极端降级 → Dollar Store、折扣店
  阶段4g 只买食物 → 非食品支出崩塌
  阶段4h 食品援助 → 连食物都买不起

判断当前阶段的方法：
  看各阶段对应公司的财报，哪些已经开始恶化
```

### 4.2 麦当劳 vs Chipotle 对比

```
逻辑：
  麦当劳客群 = 低收入（客单价 $8-10）
  Chipotle客群 = 中产/白领（客单价 $12-15）

  Case 1: MCD客流跌 + CMG客流涨 → 只有低收入在砍外食（K型早期）
  Case 2: MCD客流跌 + CMG客流也开始跌 → 中产也开始省了（K型中期）
  Case 3: 两者都暴跌 → 全面衰退信号
```

## 阶段5：系统内信用压力

### 5.1 信用卡逾期率（保留但标注限制）

```
公式：逾期30天+（或90天+）的余额 ÷ 信用卡总余额

⚠️ 幸存者偏差机制：
  1. 银行减少向次级发卡 → 分母中次级占比下降
  2. 银行主动降低次级额度 → 次级余额缩小
  3. 次级借款人破产/销卡 → 从分母中消失
  → 结果：逾期率"变好"，但实际情况在恶化

修正方法：
  ├─ 看绝对逾期金额（不看比率）
  ├─ 看次级专营商（Synchrony、Bread Financial）数据
  └─ 结合阶段6（准入率）判断分母缩小了多少
```

### 5.2 净核销率（Net Charge-Off）

```
公式：(核销金额 − 回收金额) ÷ 平均贷款余额

为什么比逾期率更可靠：
  ├─ 逾期可以被"重组"（restructured）延迟
  ├─ 逾期可以被"下调标准"后重新分类
  └─ 核销是银行"认赔"，会计准则不允许粉饰
```

### 5.3 汽车贷款逾期率

```
公式：逾期60天+的汽车贷余额 ÷ 全部汽车贷余额

为什么汽车贷比信用卡更可靠：
  ├─ 合同签了就在系统里（银行不能单方面取消贷款）
  ├─ 分母不会因为"不发新贷"而大幅缩小
  └─ 有实物抵押品，逾期后最终会变成收回（阶段7硬数据）

扭曲风险：
  ├─ 贷款期限延长（72-84个月）→ 延迟违约暴露
  └─ 负资产（欠款 > 车价）→ 即使不逾期也是风险
```

## 阶段6：被踢出系统

### 6.1 次级信贷准入率

```
公式：新发信用卡中 FICO<660 占比

这个指标直接测量"幸存者偏差的大小"：
  ├─ 25% → 正常，银行对次级开放
  ├─ 20% → 开始收紧
  ├─ 18% → 警戒
  ├─ 15% → 系统性排斥
  └─ <12% → 接近全面关闭次级信贷

当前17.8%（前值 16.4%）：
  → (25.2% - 17.8%) / 25.2% = 约 30% 的次级借款人被排除
  → 这约 30% 的人去了哪里？BNPL、EWA、当铺、家人借款
```

## 阶段7：硬资产损失

### 7.1 汽车收回量

```
为什么是"最诚实"的指标：
  ├─ 物理事件，无法统计美化
  ├─ 用绝对数量（不是比率）→ 没有分母扭曲
  ├─ 代表借款人现金流彻底断裂
  └─ 从拍卖公司数据可交叉验证

传导关系：
  阶段5（汽车贷逾期）→ 阶段7（汽车收回）
  通常滞后 3-6 个月
```

## 阶段8-11 详解

### 8.1 破产申请

```
Chapter 7（清算型）vs Chapter 13（重组型）的意义：
  ├─ Ch.13占比高 → 人们还在"挣扎求生"，有收入但不够还债
  ├─ Ch.7占比高 → 人们"彻底放弃"，已无力偿还
  └─ Ch.7占比上升 → 情况从"挣扎"恶化到"绝望"
```

### 9.1 大小银行逾期差距

```
公式：小银行逾期率 − 大银行逾期率（或看倍数）

逻辑：
  大银行（JPM/BAC）→ 优质客户为主
  小银行/社区银行 → 服务次级和本地社区

差距扩大 = K型分化加剧
  1.0x → 无分化
  1.5x → 正常差距
  2.0x → 警戒
  2.5x+ → 严重分化（当前2.36x）
```

### 9.2 贷款损失准备金

```
公式：银行计提的准备金 ÷ 贷款余额

这是银行的"内部预测"：
  ├─ 银行比外部分析师更了解自己的贷款质量
  ├─ 准备金上升 = 银行预计未来损失增加
  └─ 连续3个季度上升 = 趋势性恶化确认

注意：会计准则（CECL）要求前瞻性计提，
      所以准备金变化是对未来的预判，不是对过去的记录
```

### 11.1 VIX 与信用利差

```
VIX 的含义：
  ├─ 15-20 → 市场平静
  ├─ 20-25 → 市场开始紧张
  ├─ 25-35 → 市场恐慌
  └─ 35+ → 极度恐慌（2020年3月曾到82）

高收益利差的含义：
  ├─ 300-400bp → 正常
  ├─ 500bp → 市场开始定价衰退
  ├─ 800bp → 市场定价严重衰退
  └─ 1000bp+ → 金融危机级别（2008年达2000bp）

关键：当前VIX=15-18 + 利差=350-400bp
     说明市场尚未定价前10个阶段的恶化
     这种"基本面vs市场"的背离历史上最多持续6-18个月
```
